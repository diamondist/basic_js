Дан код:
var a = 1, b = 1, c, d;
Почему код даёт именно такие результаты?

c = ++a; alert(c);           // 2
Выполняется сначала префиксный инкремент, результат присваивается переменной с.

d = b++; alert(d);           // 1
В постфиксной форме инкремента сначала происходит присваивание, затем увеличение переменной.

c = (2+ ++a); alert(c);      // 5
Сначала происходит инкремент переменной а, она становится равной 3, затем сложение

d = (2+ b++); alert(d);      // 4
Сначала происходит сложение, затем инкремент b.

alert(a);                    // 3
alert(b);                    // 3
В результате всех предыдущих строк обе переменные и a и b дважды подверглись операции инкремента, соответственно теперь равны 3.
